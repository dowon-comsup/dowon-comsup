# [공지] 도원컴섭(dowon-comsup) 자동화 포털 현황 안내

**작성일:** 2026년 4월 13일  
**대상:** 기업지원팀 전 팀원

---

## 1. 개요

도원컴섭은 **노무법인 도원 고용지원팀**의 업무 자동화를 위한 웹 기반 도구 모음입니다. 고용장려금 신청에 필요한 서류 검증, OCR 추출, 보험 판정 등의 반복 업무를 브라우저에서 직접 처리할 수 있도록 만들어졌습니다.

**핵심 특징:**
- 모든 처리가 **브라우저 내에서** 이루어지며, 서류가 외부 서버로 업로드되지 않습니다
- AI Vision(Gemini, Claude)을 활용한 스캔 서류 자동 판독
- 별도 프로그램 설치 없이 웹 브라우저만으로 사용 가능
- GitHub Pages를 통해 배포되어 있어 언제든 접속 가능

## 2. 현재 보유 도구 (총 5종)

### (1) 급여서류 검증도구 v8.6
- **용도:** 고용장려금 신청 시 급여대장, 이체확인서, 급여명세서 간 교차 검증
- **주요 기능:**
  - PDF 급여대장 자동 파싱 및 기간별 완전성 확인
  - 이체금액과 급여명세서 간 금액 일치 여부 검증
  - 대상 근로자 포함 여부 자동 체크
  - 누락 서류 자동 식별
  - 검증 결과 등급 산정 (A/B/C/D) 및 엑셀 내보내기
- **지원 장려금:** 청년일자리도약장려금, 육아기 근로시간 단축 지원금 등
- **AI 처리:** Gemini 2.5 Flash (1차) → Claude Sonnet (Gemini 오류 시 자동 전환)

### (2) 청년일자리도약장려금 서류 판독
- **용도:** 청년일자리도약장려금 신청 서류 6종의 일괄 판독 및 수급 자격 판정
- **검증 서류:** 근로계약서, 졸업증명서, 사업자등록증명, 학력확인서, 동의서 등
- **주요 기능:**
  - AI를 통한 서류 자동 분류 및 내용 추출
  - 정규직 여부, 연령/학력 요건, 기업 규모, 기수급 이력 등 자격 요건 자동 판정
  - 복수 신청자 동시 처리 지원
  - 예상 장려금 금액 자동 산출

### (3) 고령자 계속고용 지원금 판독
- **용도:** 취업규칙에서 정년을 추출하고, 정년 도래 근로자를 자동 식별
- **주요 기능:**
  - 취업규칙 PDF(스캔본 포함)에서 정년 조항 자동 추출
  - 근로자 명부(엑셀/CSV/PDF)와 대조하여 정년 도래자 식별
  - 향후 3년간 정년 도래 예정자 선제 파악
  - 중복 검출 및 재직 상태 필터링

### (4) 등기임원 4대보험 판정
- **용도:** 등기임원의 4대 보험(고용/산재/건강/국민연금) 가입 적정성 판단
- **주요 기능:**
  - 법인 등기부등본에서 임원 정보 자동 추출
  - 보험 가입 현황과 대조하여 취득취소/상실신고/정상 분류
  - 대표이사 이력 및 친족관계 추적
  - 보험료 소멸시효(3년) 자동 계산

### (5) 수기서류 OCR 추출
- **용도:** 손글씨로 작성된 스캔 서류에서 텍스트를 자동 추출
- **주요 기능:**
  - Gemini / Claude Vision 이중 지원
  - 병렬 페이지 처리로 대량 서류 빠른 처리
  - 사업장정보, 진단기초정보, 위임장, 대리인선임신고서 등 양식별 구조화 추출
  - 해상도 조절(2x~3x) 기능으로 인식률 향상

## 3. 사용 방법

### 접속
- GitHub Pages 배포 주소를 통해 접속하면 대시보드가 표시됩니다
- 대시보드에서 원하는 도구를 선택하여 진입합니다

### API 키 설정
- AI 기능을 사용하려면 **최초 1회** API 키를 입력해야 합니다
  - **Google Gemini API Key** — 급여검증, OCR 추출에 사용
  - **Anthropic Claude API Key** — 서류 판독, 임원 판정, OCR에 사용
- 입력한 키는 **본인 브라우저의 localStorage**에만 저장되며, 외부로 전송되지 않습니다
- 한 번 입력하면 같은 브라우저에서는 재입력 불필요합니다

### 기본 사용 흐름
1. 해당 도구 페이지 접속
2. API 키 입력 (최초 1회)
3. PDF/엑셀 등 서류 업로드
4. AI 분석 실행 → 결과 확인
5. 필요 시 엑셀 내보내기 또는 수동 검토

## 5. 활용 방안

### 즉시 활용 가능한 업무
- **장려금 신청 전 사전 검증:** 급여서류 검증도구로 제출 전 누락/불일치 사전 확인
- **대량 신청 건 처리:** 복수 신청자 일괄 판독으로 건별 수작업 시간 대폭 절감
- **정년 도래자 사전 파악:** 고령자 계속고용 도구로 향후 3년 대상자를 선제 관리
- **등기임원 점검:** 신규 위탁 사업장의 임원 보험 적정성을 빠르게 판단
- **손글씨 서류 디지털화:** OCR 도구로 수기 작성 서류를 구조화된 데이터로 변환

### 업무 효율화 효과
- 급여서류 교차 검증: 수작업 대비 **처리 시간 대폭 단축**
- 서류 누락/오류 사전 발견으로 **반려율 감소**
- 정형화된 판정 기준 적용으로 **판단 일관성 확보**

## 6. 유의사항

1. **개인정보 보호:** 모든 처리가 브라우저에서 이루어지지만, AI API 호출 시 서류 이미지가 API 제공사(Google/Anthropic)로 전송됩니다. 민감한 개인정보가 포함된 서류 처리 시 이 점을 인지해 주세요.
2. **API 사용량:** AI 기능 사용 시 API 토큰이 소비됩니다. 급여검증도구에는 사용량 추적 기능이 있으니 참고해 주세요.
3. **결과 검증:** AI 판독 결과는 참고용이며, 최종 판단은 반드시 담당자가 확인해야 합니다.
4. **브라우저 호환:** Chrome 또는 Edge 최신 버전 사용을 권장합니다.
